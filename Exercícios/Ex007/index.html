Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito usando a tag <b> (Não Semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (forma Semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (Não Semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (Semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande / Pequeno

Estamos criando um texto Grande e um texto Pequeno nesse parágrafo, sendo usando a tag <big> para deixa o texto Grande (Não Semântica) e para o texto Pequeno a tag <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois para fazer isso basta usar a tag <ins> (Semântica).

Existe também o sublinhado com a tag <U> (Não Semântica).

Texto Sobrescrito

Serve para inserir coisas do tipo x20+3 utilize a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H2O é a tag <sub>.
